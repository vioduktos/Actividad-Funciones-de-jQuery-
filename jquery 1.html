FUNCIONES

.CLICK

El evento de clic ocurre cuando se hace clic en un elemento.
en este ejemplo al presionar el botón, tiene que decir "apretaste click"

.HIDE

El método hide () oculta los elementos seleccionados.
en este ejemplo al presionar el boton, ocultará el div

.SHOW

El método show () muestra los elementos seleccionados ocultos.
en este ejemplo al presionar el botón, se mostrará el div oculto

.TOGGLE

El método toggle () alterna entre hide () y show () para los elementos seleccionados.
en este ejemplo al presionar el botón se alternará con el párrafo

.SLIDEUP

El método slideUp () desliza hacia arriba (oculta) los elementos seleccionados.
en este ejemplo al presionar el botón se ocultará el título

.SLIDEDOWN

El método desliza hacia abajo (muestra) los elementos seleccionados.
en este ejemplo al presionar el botón se mostrará el título de SlideUP

SLIDETOGGLE

El método alterna entre slideUp () y slideDown () para los elementos seleccionados.
en este ejemplo al presionar el botón, se alternará el título

.FADEIN

El método jQuery se utiliza para fundir(mostrar) un elemento oculto.
antes de presionar el botón oculta hide y título de slideToggle

.FADEOUT

Se usa para desvanecer un elemento visible.


en este ejemplo al presionar el botón se desvanecerá el div click, show y hide

.ADDCLASS

El método addClass () agrega uno o más nombres de clase a los elementos seleccionados.
en este ejemplo al presionar el botón las cajas verdes tomaran la clase de las amarillas

.BEFORE

El método before () inserta el contenido especificado delante (antes) de los elementos seleccionados.
en este ejemplo al presionar el botón se insertará antes del título " funciona"

.AFTER

El método after () inserta el contenido especificado después de los elementos seleccionados.
en este ejemplo al presionar el botón se insertará después del parrafo "vioduktos rapers"

.APPEND

El append()método agrega un elemento al final de la lista.

  • vioduktos
  • sabado

en este ejemplo al presionar el botón se añadirá a la lista "carrete el sabado"

.HTML

El método html () establece o devuelve el contenido (innerHTML) de los elementos seleccionados.
en este ejemplo al presionar el botón se agregará otro título

.ATTR

El método attr () establece o devuelve atributos y valores de los elementos seleccionados.
en este ejemplo al presionar el botón las vieñetas de la lista del div append se transforman a cuadrado

.VAL

El método val () devuelve o establece el atributo de valor de los elementos seleccionados.Se usa principalmente con elementos de formulario.

Nombre:


en este ejemplo al presionar el botón se establecerá un nombre

.TEXT

El método text () establece o devuelve el contenido de texto de los elementos seleccionados.
en este ejemplo al presionar el botón todos los textos de los botones dirán "presionar"
